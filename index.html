HTML

  • HTML or Hypertext Markup Language
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS describes how elements should be rendered on screen, on paper, in speech, or on other media
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS, and extenal CSS style sheet
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • git checkout branch-name: switch to another branch
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: fetch and merge any commits from the tracking remote branch
  • git push [alias] [branch]: transmit local branch commits to the remote repository branch

JavaScript

  • JavaScript is a powerful programming language that can add interactvity to a website.
  • JavaScript is a dynamic language with dynamic types
  • Variable is a named container that allow us to store data in our code
